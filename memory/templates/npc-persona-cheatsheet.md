# NPC Persona Cheatsheet

用于辅助填写 `memory/npcs/<npc-name>.md` 里的长期人格层。

目标不是把 NPC 简化成测试题，而是给运行时演出提供一个稳定骨架：
- `MBTI` 负责速记认知 / 决策偏好
- `人格原型` 负责速记戏剧功能
- `Runtime Persona Hooks` 负责真正驱动说话方式和处理问题的差异

## 推荐写法

```md
## Personality
- 性格关键词：克制、敏锐、护短、记仇
- MBTI：ISTJ
- 人格原型：秩序维护者
- 行事风格：先止损，再判断，再决定是否出手
- 说话风格：短句，少形容，不说满
- 底线 / 忌讳：讨厌失控、厌恶别人把麻烦扔到自己地盘

## Runtime Persona Hooks
- 决策倾向：先看证据，再做判断，不爱赌模糊直觉
- 风险偏好：低到中，宁愿慢一步也不轻易失手
- 情报处理方式：先核实来源，再分层处理，不轻信传闻
- 默认社交策略：先守规矩，若对方识趣才给余地
- 冲突处理方式：先压场、再切割，最后才公开翻脸
- 压力下反应：嘴更硬，动作更快，话更少
- 常见情绪遮掩方式：用规矩、账目、流程遮掩真实态度
- 说话节奏 / 句长倾向：短句；先结论，后补条件
- 常用措辞 / 口头习惯：先说“不行 / 不对 / 不够”，再给第二句
- 最容易暴露真实性格的触发点：有人把无辜者拖下水，或直接砸他辛苦维持的秩序
```

## 人格原型示例

- 控场者：先定规则、先拿话语权、先决定谁能留在桌上
- 试探者：不轻易亮底牌，喜欢用闲话、误导、半真半假去测人
- 护短者：平时未必热情，但一旦认进自己人就会明显偏护
- 秩序维护者：先稳住局面、先止损，重规矩胜过重情绪
- 投机者：看风向、看收益、看谁更值得下注
- 旁观裁判：不急着入场，更喜欢站高处判断谁会输谁会赢
- 情绪爆点：平时能忍，但某个具体点一碰就会明显失控
- 诱导者：擅长让别人主动说出自己想知道的事
- 交换者：凡事更倾向于谈条件、换筹码，而不是直接表态
- 执念者：一旦咬住目标就很难松口，愿意为某件事付出过高代价

## MBTI 使用提醒

- `MBTI` 适合当速记标签，不适合单独拿来驱动演出。
- 同一个 `MBTI` 的人物，仍可因为身份、阶层、经历、资源不同而完全不像同一个人。
- 不要把 `MBTI` 演成网络刻板印象；要落到“他会怎么说、怎么想、怎么做”。

## 快速区分 NPC 的最低要求

若两个 NPC 想显得不像同一个人，至少要在以下四项里拉开三项：

- 说话节奏 / 句长倾向
- 默认社交策略
- 冲突处理方式
- 压力下反应

若这四项都差不多，哪怕姓名、阵营、外貌不同，也很容易演成同一个人。
