## 填写规则
- 只给会反复出场、能影响多轮剧情的 NPC 建档。
- 不要把每次临场对话都塞进档案；只沉淀长期有效的信息。
- 若某条内容只是传闻、误判或未证实事实，应明确标记，不要当成确定真相写死。
- `MBTI` 用来帮助区分人物的认知与决策方向，但必须落到“决策倾向 / 风险偏好 / 说话节奏 / 冲突处理方式”等可演出字段里。
- 若两个 NPC 的 `MBTI` 相同，也必须通过身份、阶层、经历、资源与运行时 persona hooks 把他们拉开，不要直接演成同一个人。
