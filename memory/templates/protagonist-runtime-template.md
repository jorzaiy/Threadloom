# Protagonist Runtime Template

用于 `state.md -> Protagonist Runtime` 区块。
主角长期设定以 `player-profile.md` / `player-profile.json` 为准，这里只维护当前运行态。

```md
- 主角：
- 当前状态：
- 当前位置：
- 最近公开露面：
- 最近确认位置：
- 当前同行者：
- 当前伤势 / 消耗：
- 当前显性压力：
- 当前隐藏压力：
- 当前持有关键物件：
- 已公开暴露的信息：
- 仍属私密的信息：
```

## 填写规则
- `当前状态` 只写本轮会直接影响演出的状态，例如“潜逃中 / 养伤中 / 刚入局 / 与某人对峙”。
- `最近公开露面` 和 `最近确认位置` 用于追踪、搜捕、堵截类事件，不要省略。
- `当前同行者` 只写当前同场或同路的重要角色。
- `当前伤势 / 消耗` 只写会影响行动判定的部分。
- `已公开暴露的信息` 与 `仍属私密的信息` 用于防止信息边界混乱。
