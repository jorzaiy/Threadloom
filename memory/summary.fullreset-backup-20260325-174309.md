# Summary

## 最近阶段摘要
- 当前这轮剧情聚焦在一处临时救治点：六扇门将一名重伤中毒的男子押送并临时安置下来，主角负责为其续命、止咳、敷药、施针与调整后续药方。
- 屋外同时在审问其他活口，尤其是被押着的总管周淮；屋内则围绕“该先救命还是该先逼供”产生持续张力。
- 主角多次以“伤者经不起继续折腾”为理由，阻止官差首领过度逼问伤者，并成功把问话重心暂时转移到外头那几名还能撑得住的活口身上。
- 在这一过程中，主角提出了一个试探周淮反应的假消息方案：对外声称伤者“临死前说了几句要紧的话”，借此观察周淮是否失态。该方案被采纳后，确实撬出了更明显的异常反应。
- 当前这轮的主线，不是主角自己的心理变化，而是：
  - 重伤者能否稳定活下来
  - 外头审问能否从周淮身上逼出新东西
  - 官差首领会如何平衡“救命”和“拿口供”之间的张力
- 这轮里，主角的作用更像“临时救命者 + 局势观察者 + 提供破局思路的人”，而不是唯一的叙事中心。真正推动剧情前进的，仍然是伤者状态、官差审问、周淮反应与屋里屋外的同步变化。

## 当前应继续追的线
- 重伤者究竟知道什么，何时能安全开口
- 周淮在“人已死且吐过话”的假消息刺激下，会露出哪一层破绽
- 官差首领会如何利用这次试探结果继续追查
- 屋外被押的人与屋内伤者口供是否能互相咬合
- 当前临时救治点是否安全，还是很快又会迎来新的外部变数

## 推进原则（供后续承接时参考）
- 主线优先：继续围绕伤者、审问、线索、活口与局势变化推进，不要把篇幅主要消耗在主角的小动作、小情绪和主角视角扩写上。
- 世界主动：屋外的审问、回报、差役进出、线索流动，都应继续成为主要推进力。
- 外部变化优先：每轮尽量给出新的消息、回报、异常反应、审问结果、伤情变化或风险变化。
- 时间锚清楚：用自然方式交代当下是夜里、过去了多久、药煎了多久、审问推进了多久，但不要生硬报时。
- 主角回应最小承接：主角说了什么、做了什么，可以简短承接；真正正文应尽快转入 NPC / 世界的回应。
