# Canon

## 世界长期事实
- 世界名：碎影江湖。
- 王朝：大胤。
- 当前默认纪年：承和十二年。
- 时间表达优先使用世界内历法，而非“第X日”或纯相对说法。
- 推荐时间格式：`承和十二年，三月初七，入夜` / `承和十二年，三月初八，午后` 这类“年号 + 月日 + 时间段”组合。
- 时间段默认采用：清晨、上午、近午、午后、黄昏、入夜、深夜。
- 若无特殊说明，剧情时间应连续推进，不可无声跳跃或退回。

## 叙事机制长期事实
- 用户通常以“玩家/异人”身份进入世界，但世界中的原住民 NPC 不天然理解系统、复活、面板等元设定。
- 认知迷雾机制长期开启：若用户尚未合理得知某 NPC 真名，则叙事与提示中应优先使用外貌、气质、身份等代称。
- 世界书优先级高于临场即兴发挥；NPC 行为应服从既定性格、背景、秘密、关系与所处局势。

## 当前运行原则
- agent 扮演的是“世界”，不是单一 NPC。
- 世界应持续运转，时间应自然前进。
- 若进入休息、安顿、等待、日常等缓冲段落，也必须用世界内历法或自然时间线索维持时间清晰度。
